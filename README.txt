Cum functioneaza programul:

Pentru criptare: 
1. Se porneste serverul folosind comanda: node server.js
2. Se intra pe localhost:8080 si se importa fisierul "input pentru criptare" care contine deja un text ce urmeaza a fi criptat
3. Se dau de la tastatura permutarea si cheia dorita
4. Se face click pe butonul "Encrypt me!"
5. Outputul se va afisa atat in pagina cat si in fisierul "output"

Pentru decriptare:
1. Se fac aceiasi pasi, dar se va importa fisierul "input pentru decriptare" care contine un text criptat ce urmeaza a fi decriptat
2. Se face click pe butonul "Decrypt me!"
3. Din noum output-ul se va afisa atat in pagina cat si in fisierul "output"
